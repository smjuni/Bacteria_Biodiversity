Synopsis

This project identifies the prevalent bacteria found in each subject's navel. To utilize this dashboard, please choose your ID (each subject was given a unique ID number at registration). Use this number to access your results.

The bar chart shows the top ten bacteria found in each subject's navel as well as the bacteria ID number and type.
The bubble chart visualizes the amount of each bacterium found in each subject's belly button.
Finally, the gauge chart marks the number of times per week the subject washes their navel.
